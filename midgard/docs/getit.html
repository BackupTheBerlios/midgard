How to get it (LINUX)

Quellen als TGZ-Archiv:
  • Zur Zeit nicht verfügbar, da sich der Quellcode zu schnell verändert.
CVS (Current Versions System):
  1. Zunächst benötigt man das CVS-Skript (Link speichern).
  2. Ein Verzeichnis anlegen, in welchem MAGUS compiliert werden soll und hinein wechseln. Das CVS-Skript ebenfalls in das Verzeichnis kopieren.(ggf mit 'chmod 700 midgard_cvs' ausführbar machen)
  3. Benötigte Dateien herunterladen:
    1. Online gehen ;-)
    2. 'midgard_cvs checkout' eingeben (Passwortabfragen mit 'ENTER' quittieren). Abwarten, bis der Download beendet ist.
    3. Offline gehen

  4. Kompilieren des Sourcecodes:

    1. 'midgard_cvs install' eingeben. Zur systemweiten Installation wird das root-Passwort abgefragt
    2. Jetzt sollte die Eingabe von 'midgard' dazu führen das der MCG gestartet wird.

  5. Updates:

    1. Online gehen ;-)
    2. 'midgard_cvs update' (in dem Verzeichnis, von dem aus auch der 'checkout' stattgefunden hat) eingeben. Es werden nur geänderte/neue Dateien heruntergeladen
    3. Offline gehen
    4. mit 'midgard_cvs install' die (geänderten/neuen) Quellen wieder übersetzen
Binary (rpm)
  • Zur Zeit gibt es noch kein RPM-Paket
Binary (Tarball (tar.gz))
  • Eine fertig kompilierte Version findet sich unter ftp://ftp.berlios.de/pub/midgard/bin/
    Die vorliegende Version wurde unter SuSE 7.2 (mit einigen Updates) kompiliert, so dass nicht sicher ist, ob sie auch mit einer anderen Konfiguration funktionsfähig ist.
    Diese Version soll später durch ein rpm-Packet ersetzt werden.
    Die Datei ist ca. 6 MB groß, und muß nur in ein beliebiges Verzeichnis entpackt werden.

How to get it (Windows)

  • Download:
     Das Zip-Archiv findet sich unter: ftp://midgard.berlios.de/pub/midgard/Win32/magus.zip (ca. 3.5 MB)


  • Installation:
    Das heruntergeladene Zip-Archiv in ein Verzeichnis entpacken.
    Es werden keine Eintragungen in die Registry vorgenommen.
    Die Datei 'magus.exe' ist bereits das ausführbare Programm
