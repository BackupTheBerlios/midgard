Allgemeine Voraussetzungen

  • Windows 95/NT (oder neuer)
  • ca. 50MB Platz auf der Festplatte
  • mindestens eine Bildschirmauflösung von 800x600 Pixel
  • Einen PDF-Viewer (z.B. AcrobatReader)

ZIP-Archiv

Um das ZIP-Archiv zu entpacken wird zusätzlich ein (Ent-)Packer, wie pk[un}zip (DOS), WinZip oder Totalcommander benötigt.
