1.1 Voraussetzungen (LINUX)

Der Midgard Charakter Generator (MCG) benötigt:
  1. [als Datenbankserver Postgresql ab Version 7.1] zur Zeit läuft die Umstellung auf xml
  2. Das Current Versions System (CVS) - sollte bei jeder Distribution vorhanden sein
  3. einen c++ Compiler (solange es noch kein rpm-Paket gibt, hat bei jeder Distribution dabei zu sein)
  4. Gtk+ (ab Version 1.2.8)
  5. Gtk-- / Gtkmm (ab Version 1.2.4)
  6. LaTeX
  7. Ghostview
  8. libsigc++
  9. Für Mitentwickler

Während des Kompilierens werden ca. 210 MB auf der Festplatte benötigt, sowie ~250 MB Arbeitsspeicher.
Allerdings läßt sich dies reduzieren bzw. umgehen..
Der fertige MCG belegt ca. 35 MB auf der Festplatte. Zurück

1.2 Voraussetzungen (Windows)
  1. Windows95 (oder neuer)
  2. ca. 10 MB Platz auf der Festplatte
  3. Die Bildschirmauflösung sollte wenigstens 800x600 Pixel betragen
