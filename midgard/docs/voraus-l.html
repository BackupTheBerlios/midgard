1.1 Voraussetzungen (LINUX)


Die Linux variante von MAGUS benötigt:
  1. als Datenbankserver Postgresql ab Version 7.1
    Zur Zeit läuft die Umstellung auf xml. Wenn die Umstellung vollständig abgeschlossen ist, sollte Postres nicht mehr notwendig sein.
  2. Das Current Versions System (CVS) - sollte bei jeder Distribution vorhanden sein.
    Dies gilt zumindest solange, wie es kein .tgz, .rpm oder .deb Archiv gibt. Zur Zeit gibt es zu häufige Änderungen, als das sich das Packen eines (Binary-) Archives lohnen würde.
  3. einen c++ Compiler
    (solange es noch kein rpm-Paket (bzw. fertiges binary) gibt, hat bei jeder Distribution dabei zu sein)
  4. Gtk+ (ab Version 1.2.8)
  5. Gtk-- / Gtkmm (ab Version 1.2.4)
  6. LaTeX
  7. Ghostview
  8. libsigc++
  9. Für Mitentwickler

Während des Kompilierens wird unter Umständen übermäßig viel Arbeitsspeicher benötigt. Falls es hier Probleme gibt sich dies folgendermaßen beheben bzw. umgehen..
Der fertige MAGUS belegt ca. 15 MB auf der Festplatte.
