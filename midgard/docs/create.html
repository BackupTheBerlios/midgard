Abenteurer erschaffen

Es gibt grundsätzlich zwei Möglichkeiten, einen neuen Abenteurer zu erschaffen.
  1. Einen Abenteurer gemäß Regelwerk 'auswürfeln'
  2. Eine Figur von Hand eingeben

I. Die Grundwerte:

Eine Beschreibung der Bedienelemente findet sich auf diesen Seiten.

Einen Abenteurer gemäß Regelwerk 'auswürfeln'

  1. Zunächst wird die Spezies ausgewählt. (Voreingestellt ist hier 'Mensch')
    Bei Elfen wird durch eine Abfrage zusätzlich darüber entschieden, ob es ein 'Doppelabenteurer' werden soll.
  2. Auswürfeln der Basiseigenschaften (Grundwerte würfeln)
    Es gibt drei Varianten, wie die Basiseigenschaften bestimmt werden können:
    (Alle drei Möglichkeiten werden über den Button 'Grundwerte würfeln'ausgeführt)

    1. Es wird 2x pro Eigenschaft gewürfelt, das höhere Ergebnis wird übernommen (standard)
      Diese Variante wird ausgeführt, wenn der Button mit der linken Maustaste betätigt wird.
    2. Es wird 2x gewürfelt, der bessere Wert wird einer Eigenschaft zugewiesen. Diese Prozedur wird 6x durchgeführt, bis alle Eigenschaftswerte bestimmt sind.
      Diese Variante wird über die mittlere Maustaste realisert. Die Zuordnung erfolgt über Buttons am unteren Rand des MAGUS-Fensters.
    3. Es wird 9x gewürfelt, und die 6 besten Werte werden den Eigenschaften zugewiesen. (6 von 9)
      Diese Variante wird mit der rechten Maustaste gewählt. Die Zuordnung der Werte erfolgt analog zu b.

    Da die mittlere Maustaste unter Windows nicht einheitlich unterstützt wird, existieren links neben dem Button 'Grundwerte würfeln' drei Buttons, die diese Funtktionen zugänglich machen.
    Alle Werte können mittels 'Werte editieren' verändert werden.
  3. Das Geschlecht wird ausgewählt (Voreingestellt ist 'männlich')
  4. Der Abenteurertyp wird gewählt (Die Voreinstellung ist abhängig von der Spezies, bei Menschen ist es der 'Assassine')
  5. Die Herkunft auswählen (Stadt oder Land, abhängig vom Abenteurertyp)
  6. Die abgeleiteten Werte werden ausgewürfelt.
    Auch diese lassen sich mittels 'Werte editieren' verändern. Nach einer Änderung muß(!) der Button 'Werte für Grad 1 setzen' betätigt werd, damit die Änderungen in die Berechnung der Eerfolgswerte einbezogen werden können.
  7. Spätestens jetzt sollten mittels 'Werte editieren' der Name des Abenteurers und die Version in die entsprechenden Felder eingegeben werden. Ausserdem kann ein passender Glaube eingegeben werden. (Die Eingaben lassen sich später noch ändern, in früheren Versionen von MAGUS wurden die Daten des Abenteurers unter dem eingegebenen Namen und Version gespeichert)


Weiter geht es mit den Lernschemata

Einen Abenteurer direkt eingeben:

  1. Sinnvollerweise werden zunächst Spezies, Abenteurertyp Geschlecht und Herkunft ausgewählt.
  2. Im nächsten Schritt werden nach der Aktivierung des Buttons 'Werte editieren' die gewünschten Werte in die jeweiligen Felder eingetragen. Es sollten jedoch nur Werte für den 1. Grad eingegeben werden, da sonst einige Erfolgswerte nicht korrekt berechnet werden können
  3. Zum Abschluß wird der Button 'Werte für Grad 1 setzen' betätigt.


Weiter geht es mit den Lernschemata

II. Die Lernschemata:

Eine Beschreibung der Bedienelemente findet sich auf diesen Seiten.
  1. Zunächst wird das Herkunftsland gewählt.
  2. Im nächsten Schritt wird eine angeborene Fertigkeit bestimmt:
    1. Mit der linken Maustaste wird eine (oder je nach Würfelglück auch eine zweite) angeborene Fertigkeit ausgewürfelt
    2. Über die rechte Maustaste erreicht man ein Auswahlfenster, in welchem eine oder mehrere angeborene Fertigkeiten ausgewählt werden können
  3. Als nächstes erfolgt die Bestimmung der Lernpunkte:
    1. Mit dem Button 'Lernpunkte würfeln' werden die Lernpunkte für alle Fertigkeitskategorien ausgewürfelt.
    2. Der Button 'Lernpunkte editieren' ermöglicht es den einzelnen Fähigkeitskategorien Lernpunkte zuzuordnen, oder bereits erwürfelte Lernpunkte zu verändern.
    3. Die beiden Pfeilbuttons unterhalb der Lernpunkte ermöglichen es, Lernpunkte von den 'allgemeinen Fertigkeiten' und den 'Waffenfertigkeiten' auf die Lernpunkte für 'ungewöhnliche Fertigkeiten' zu übertragen.
  4. Magier wählen ihr Spezialgebiet aus. Bei anderen Abenteurertypen ist diese Option nicht sichtbar!
  5. Fertigkeiten erwerben:
    Durch das Betätigen der Buttons für die einzelnen Fertigkeitskategorien (jeweils unter den dazugehörigen Lernpunkten) werden die Fertigkeiten auswählbar gemacht:
    Im rechten Fenster werden die erlernbaren Fertigkeiten angezeigt. Die gewünschte Fertigkeit wird mit der Maus selektiert und automatisch zu den bereits erlernten Fertigkeiten im linken Fenster angezeigt.
    Bei magisch begabten Abenteurern erfolgt hier auch die Auswahl der Zauber.
  6. Abhängig vom Abenteuerertyp kann über den entsprechenden Button eine Spezialwaffe ausgewählt werden.
  7. Die Auswahl des Berufes kann auf zweierlei Art erfolgen:
    1. Wird die rechte Maustaste benutzt, kann eine Berufsbezeichnung direkt eingegeben werden. Dabei werden jedoch keine berufspezifischen Fertigkeiten berücksichtigt
    2. Mit der linken Maustaste wird ein Wert für den Beruf ausgweürfelt.
      Es öffnet sich ein Fenster mit einer Liste der erlernbaren Berufe und den zugehörigen Fertigkeiten:
      Durch die Auswahl einer Fertigkeit wird der dazugehörige Beruf ausgewählt.
      Der Wert des Berufwurfes kann über den Button 'Würfeln' wiederholt, bzw. direkt bestimmt werden:
      1. mit der linken Maustaste wird der Würfelwurf wiederholt
      2. wird die rechte Maustaste benutzt, so öffnet sich ein Eingabefeld, in welches der gewünschte Wert von Hand eingegebern werden kann.
  8. Nun folgt die Ausstattung des Abenteurers mit Geld und Waffen:
    Mit dem Anklicken des entsprechenden Buttons öffnet sich ein Fenster, in dem Geld und Waffen wahlweise ausgewürferlt oder ausgewählt werden können (Auswürfeln ist voreingestellt):
    1. Geld:
      Wird das Geld ausgewürfelt, wird der dem Wurfentsprechnde Wert 'gutgeschrieben'.
      Wird das Geld ausgewählt, so öffnet sich ein Fenster, in welchem der Wert in GS, SS und KS eingegeben werden kann.
    2. Waffen:
      Äquivalent zum Geld werden die Waffen ausgewürfelt. Abhängig vom Würfelergebnis erscheint eine Liste mit auswählbaren Waffen
      Wird 'auswählen' gewählt. so erscheint sofort die Auswahlliste.
  9. Zu guter Letzt wird die Rüstung des Abenteurers ausgewürfelt.
