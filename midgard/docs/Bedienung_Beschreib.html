Abenteurer beschreiben

Auf der Notebookseite Beschreibung bietet MAGUS die Möglichkeit, eine mehr oder weniger ausführliche Beschreibung eines Abenteurers einzugeben. Soll mehr, als ein Bild und eine unformatierte Beschreibung eingegeben werden, so ist dies durch LaTeX-Kommandos möglich. Eine kleine Übersicht über die Kommandos soll diese Seite bieten:
  • Schrift
  • Umbruch & Ausrichtung

Umbruch & Ausrichtung

Zeilenumbruch:

Syntax:

Zeile 1 \\
Zeile 2

\\ Zeilenende


Textausrichtung

Der Text wird defaultmäßig im Blocksatz gesetzt.

Syntax:

\begin {Befehl}
Text auf den der Befehl angewendet werden soll
\end {Befehl}

center zentrierter Text
flushleft linksbündiger Text
flushright rechtsbündiger Text
quote Text einrücken


Für einzelnen Zeilen gibt es folgende Alternativen:

Syntax:

\Befehl {Text}

\centerline zentrierter Text
\raggedleft linksbündiger Text
\raggedright rechtsbündiger Text

Schrift

Verändern der Schriftgröße

Syntax:

\begin{Befehl}
Text für den der Befehl gelten soll
\end{Befehl}

oder:

{\Befehl Text}

\Huge riesige Schrift
\huge extrem große Schrift
\LARGE sehr große Schrift
\Large ziemlich große Schrift
\large große Schrift
\normalsize normal große Schrift
\small kleine Schrift
\footnotesize kleinere Schrift
\scriptsize sehr kleine Schrift
\tiny winzige Schrift


Schriftart

Syntax:

\Befehl { Text für den der Befehl gelten soll }

\textrm Roman (Serifen)
\textsf Sans Serif
\texttt Typewriter (Courier)


Schriftform

Syntax:

\Befehl { Text für den der Befehl gelten soll }

\textbf Fett
\textit Italic
\textsc Kapitälchen
