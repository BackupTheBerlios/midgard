How To Get It



  • Linux
  • Windows
  • XML-Daten
Es gibt mehrere Möglichkeiten, die Windows-Version von Magus zu bekommen:
  • ZIP-Archiv
  • Selbstenpackendes Archiv
  • Einzeldateien
  • Updates
  • Aus dem CVS
Zum Seitenanfang

ZIP-Archiv

Die offizielle Release des zip-Archivs findet sich unter:
http://developer.berlios.de/project/filelist.php?group_id=90
Aktuellere, aber nicht unbedingt stabile Versionen finden sich auf dem FTP-Server:
ftp://midgard.berlios.de/pub/midgard/Win32/
in der Form "magus_[VERSIONSNR]_w[NR].zip"
Zum Seitenanfang

Selbstenpackendes Archiv

Das selbstentpackende Archiv ist identisch mit dem ZIP-Archiv (die entsprechende Versionsnummer vorausgesetzt). Es lässt sich jedoch ohne eine seperates Packprogramm entpacken.
Das selbstenpackende Archiv findet sich auf dem FTP-Server unter;
ftp://midgard.berlios.de/pub/midgard/Win32/
als Datei mit dem Namen: "magus_[VERSIONSNR]_w[NR]_full.exe"
Zum Seitenanfang

Einzeldateien

Es besteht ebenfalls die Möglichkeit, Magus in Form von Einzeldateien herunterzuladen. Diese Dateien stehen unter
ftp://midgard.berlios.de/pub/midgard/Win32/magus
Für eine Installation sind alle Dateien, einschließlich der vollständigen Unterverzeichnisse notwendig.
Zum Seitenanfang

Updates

Um bei einer neueren Magus-Version nur die jeweils geänderten und/oder neu hinzugekommenen Dateien herunterzuladen und einen kommpletten Download zu vermeiden, existiert unter
ftp://midgard.berlios.de/pub/midgard/Win32/
ein Update-Paket mit dem Namen: "upgrade_[vonVERSION]_[aufVERSION|.zip
Zum Seitenanfang

CVS

Es besteht unter Windows ebenfalls die Möglichkeit, den Quellcode aus dem CVS herunterzuladen und selbst zu kompileren. (Weitere Informationen folgen)
