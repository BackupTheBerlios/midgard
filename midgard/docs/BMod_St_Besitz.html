Zum Seitenanfang

Allgemein

Die Notebookseite Besitz dient dazu, die Spielfigur mit Waffen und Rüstung(en) auszustatten.
Auf der Seite Waffen werden nur die Waffenfertigkeiten erlernt. Der Erwerb der Waffen selbst findet auf dem linken Teil dieser Notebookseite statt. Der rechte Teil dieser Seite dient zum Erwerb von Rüstungen.
Zum Seitenanfang

Buttons

Zum Erwerb von Waffen und Rüstung existieren zwei Buttons:

Magischen Bonus eingeben:
Wenn dieser Button aktiv ist, können für die ausgewählte Waffe ein Angriffs-, Schadensbonus und eine kurze Beschreibung eingegeben werden.
Rüstung:
Dieser Button existiert zweimal. Ist er aktiviert, so kann eine Rüstung aus einer Liste ausgeählt werden.
Zum Seitenanfang

Waffen


Der Erwerb der Waffen wird mittels zweier Fenster geregelt. Das obere Fenster zeigt die Waffen, die sich im Besitz der Figur befinden. Diese können mit dem Button Magischen Bonus eingeben
mit einem magischen Angriffs- und/oder Schadensbonus versehen werden, indem bei aktiviertem Button die jeweilige Waffe mit der Maus ausgewählt wird.
Eine Waffen kann "entfernt_" werden, indem sie mit der Maus einmal angeklickt wird.

Im unteren Fenster werden die Waffen angezeigt, die mit den gelernten Waffenfertigkeiten beherrscht werden, Durch einmaliges Anklicken wird eine Waffe erworben.
Zum Seitenanfang

Rüstung


MAGUS bietet die Möglichkeit zwei Rüstungen zu erwerben. Der Sinn hierfür ist, dass kaum ein Abenteurer, der eine Schwere Rüstung (z.B. Vollrüstung) besitzt, diese ständig trägt, sondern auf Reisen eine leichtere Rüstung trägt. Auf dem Dokument werden die, durch die Rüstungen modifizierten Werte, in Klammern vermerkt.

Die erste Rüstung wird gewählt, indem der erste der beiden Rüstungsbuttons aktiviert wird. Darauf hin stehen die verschiedenen Rüstungen zur Auswahl. Um eine andere Rüstung zu wählen, wir die neue Rüstung einfach mit der Maus ausgewählt.

Eine zweite Rüstung wird gewählt, indem der zweite Button aktiviert wird. Die Auswahl der Rüstung erfolgt wie oben beschrieben.
