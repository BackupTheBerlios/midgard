Die Windows-Pakete bringen fast alles mit, was MAGUS benötigt, es gibt jedoch einige wenige Kleinigkeiten, die dennoch benötigt werden:
  • Allgemeine Voraussetzungen

Allgemeine Voraussetzungen

  • Windows 95/NT (oder neuer)
  • ca. 20MB Platz auf der Festplatte
  • mindestens eine Bildschirmauflösung von 800x600 Pixel
  • Einen PDF-Viewer (z.B. AcrobatReader)

Selbstentpackendes Archiv

Die Voraussetzungen für das selbstentpackende Archiv sind mit den obigen Voraussetzungen bereits abgedeckt.

ZIP-Archiv

Um das ZIP-Archiv zu entpacken wird zusätzlich ein (Ent-)Packer, wie pk[un}zip (DOS), WinZip oder Windowscommander benötigt.
