Die Notebook-Seite "Optionen" ist in die Bereiche "globale Optionen", "Hausregeln", "Ansicht", "Drucken" und "Pfade" Unterteilt:

Originalregeln:
Ist diese Option aktiviert, so werden die Orignalregeln der 4. Auflage des Midgardregelwerks angewandt.
Ist die Option deaktiviert, können eigene Hausregeln verwendet werden. Wenn diese Option aktiviert ist, ist das offizielle Midgardlogo sichtbar.
Defaulteinstellung: aktiv
NSC zulassen::
Ist diese Option aktiviert, so können vom Spielleiter Figuren nach eigenen Ideen entworfen werden.
Defaulteinstellung: deaktiviert
Grundwerte mit nur einer Maustaste auswürfelbar machen
Ist diese Option aktiv, so können die Grundwerte mit nur einer Maustaste ausgewürfelt werden. MAGUS unterstützt drei Maustasten, unter Windwos sind jedoch nur 2 Tasten üblich und eine vorhandene dritte Taste wird nicht einheitlich unterstützt.
Defaulteinstellung: aktiviert
Wizard bei jedem Programmstart starten
Ist diese Option aktivert, wird der Wizard beim Programmstart mitgestartet.

Hausregeln:
Hausregeln werden erst auswählbar, wenn die Originalregeln deaktiviert sind.
Mit ihnen lassen sich Sonderregeln einzelner Spielrunden aktivieren.
***Hausregeln***

Drucken:
Einstellungen zu Dokumentausgabe:
MAGUS druckt die Abenteurer-Dokumente nicht direkt aus, sondern erzeugt ein, bzw. mehrere PDF-Dateien. Diese werden mit einem Viweer angezeigt und ausgedruckt.
Mit Hilfe der folgenden Optionen kann das Viewer-Programm ausgewählt werden.
Voreingestellte PDF-Viewer:
Es kann zwischen dem Acrobatreader ('acroread'), 'gv' und 'xpdf' ausgewählt werden. (nur Linux)
anderer Viewer:
Freie Viewer-Auswahl. In dieses Feld kann der bevorzugte Viewer eingegeben werden (ggf. mit Pfad)
Defaulteinstellung: Acrobatreader

Pfade und Hilfsprogramme:
In diesem Bereich werden die Pfade zu verschiedenen Verzueichnissen und Hilfprogrammen eingegeben. Es sollte der absolute Pfad angegeben werden; also unter Linux mit "/" und unter Windows mit "C:\" (bzw. dem entsprechenden Laufwerksbuchstaben) beginnen.
HTML-Browser:
Hier wird der bevorzugte Browser eingetragen, mit dem diese Hilfe aus dem Programm heraus aufgerufen wird.
Defaulteinstellung: mozilla
